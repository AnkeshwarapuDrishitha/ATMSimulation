# ATM Simulation System

A Java-based ATM Simulation project that simulates basic banking functionalities such as:

- Check Account Balance
- Deposit Money
- Withdraw Money
- Exit System

## Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Conditional Statements & Switch Case

## Features
✔ Balance Inquiry  
✔ Money Deposit  
✔ Cash Withdrawal  
✔ User-friendly Menu-driven Interface
